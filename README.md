# Pinheiro & Micho Advogados

Site institucional desenvolvido para o escritório Pinheiro & Micho Advogados, com foco em apresentar a história, estrutura, áreas de atuação, equipe jurídica, notícias e informações de contato.

## Sobre o projeto

O projeto tem como objetivo criar uma presença digital profissional, moderna e responsiva para o escritório de advocacia, oferecendo uma navegação simples e elegante para os visitantes.

O site conta com seções informativas, layout sofisticado, identidade visual em tons de azul escuro, dourado e branco, além de adaptação para dispositivos móveis.

## Funcionalidades

- Página inicial com imagem de destaque
- Menu de navegação fixo
- Menu lateral responsivo para dispositivos móveis
- Seção sobre a história do escritório
- Seção de estrutura do escritório
- Áreas de atuação jurídica
- Destaque para Direito Previdenciário
- Apresentação dos advogados
- Notícias e artigos jurídicos
- Formulário de contato visual
- Botões sociais para Instagram, Facebook e X
- Layout responsivo para desktop, tablet e celular

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Vercel

## Responsividade

O site foi ajustado para funcionar corretamente em diferentes tamanhos de tela, incluindo computadores, tablets e celulares.

Entre os ajustes realizados estão:

- Menu mobile com sidebar
- Imagens adaptáveis
- Cards reorganizados em telas menores
- Botões ajustados para mobile
- Seções em coluna única no celular
- Correção de estouro lateral
- Barra social adaptada para dispositivos móveis

## Deploy

O projeto foi publicado utilizando a Vercel.

Para realizar o deploy corretamente, o arquivo principal precisa estar nomeado como:

```txt
index.html
```

Todas as imagens utilizadas no projeto devem estar na mesma pasta do arquivo principal, ou os caminhos devem ser ajustados no HTML e CSS.

## Como executar localmente

1. Clone o repositório:

```bash
git clone https://github.com/Provadelli/Site-Pinheiro-Micho-Advogados.git
```

2. Acesse a pasta do projeto:

```bash
cd Site-Pinheiro-Micho-Advogados
```

3. Abra o arquivo `index.html` no navegador.
